  • Help

    Most of the IDE's features should be pretty straight forward to use, particularly if you're already familiar with ChoiceScript. That said here are some brief details on the more important aspects.

    • The Toolbar

      There are two toolbars above the code editor. You can swap between them by using the File and Editor buttons, prefixed by the menu icon.

      One covers scene/file specific commands and the other the more general ones.

    • Running Code

      To run your code click the Run button at the start of the File prefixed toolbar. The IDE will always run your startup scene first.

    • Changing Settings

      To change settings relating to the editor (tab spacing etc) click the Settings button on the File prefixed toolbar. Click the settings button again to close the panel.

    • The Variable Tracker

      Vars will open the variable tracker. This part of the IDE will only work whilst you have code running. It lists and periodically updates all your variable values, allowing you to track them whilst play-testing. It also lists core CS variables, the changing of which is inadvisable. If you manage to break your game or just want to reset it, just click "Run" again. Click the Vars button again to close the panel.

    • Exporting Games

      Export is found under the Editor prefixed toolbar. This is an experimental feature based off of compile.html. It will attempt to compile and export all your currently open scenes as a single (and hopefully working) .html file. If one is found, it will use your *title command as the file title and html header.

    • Adding/Removing & Changing Scenes

      By default the IDE will always create the startup and choicescript_stats scenes - and will not let you delete them. In order to change, delete or add scenes, you need to be on the Editor prefixed toolbar. Your current scene is displayed in a dropdown. You should also use this to change or create new scenes. To delete your current scene, click the Delete button.

    • Saving/Loading

      Both buttons are found on the Editor prefixed toolbar. All currently active scenes are saved to your browser's local storage. If cookies/storage are disabled and/or not supported by your browser, the save will fail. If you clear your browser's data or are browsing in incognito/private mode - your save will be lost. In short, don't rely solely on this save function - always back your code up locally.

      You only have one save slot, saving again will overwrite your previous data.
