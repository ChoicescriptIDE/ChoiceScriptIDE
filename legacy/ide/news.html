Update!

You can now edit (and test code containing) multiple scenes.

Basic instructions:
  • Use the dropdown scene list to select a scene (or create a new one).

  • The name of your scene in the dropdown menu is the name you use in conjuction with *goto_scene etc.

  • Loading and saving is no longer automatic and they will save/load all scenes.

  • The Vars tracker now tracks both temp and global variables

  • Run will always run the startup scene first (for obvious reasons).
Please inform me of any bugs, but try to replicate them first! It may be a blip.
